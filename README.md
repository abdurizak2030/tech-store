# 🛒 WE TECH STORE — Modern eCommerce Website

Welcome to **WE Tech Store**, a modern, responsive, and interactive eCommerce website built using pure HTML, CSS, and JavaScript.

This project showcases a complete front-end shopping experience including product browsing, cart system, wishlist, search functionality, and responsive design.

---

## 📌 Overview

WE Tech Store is designed to simulate a real-world online shopping platform where users can:

* Browse tech products
* Add items to cart 🛒
* Save items to wishlist ❤️
* Search products instantly 🔍
* Experience smooth UI/UX across all devices 📱💻

---

## ✨ Key Features

### 🛍️ Product System

* Multiple product cards (Watches, Phones, Cameras, Speakers, etc.)
* Product images, ratings, and prices
* Hover actions (Add to Cart / Wishlist)

### 🛒 Shopping Cart

* Add items dynamically
* Cart dropdown with product list
* Auto total price calculation
* Persistent data using LocalStorage 💾

### ❤️ Wishlist

* Save favorite products
* Wishlist dropdown preview
* Real-time badge counter

### 🔍 Search Functionality

* Live search (real-time filtering)
* Instant product matching while typing
* Case-insensitive search

### 📱 Responsive Design

* Fully responsive layout
* Mobile menu with hamburger icon
* Overlay navigation system

### 🎥 Multimedia Integration

* Embedded YouTube video section
* Interactive UI components
